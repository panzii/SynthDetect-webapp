Abstract

In the realm of computer vision, discerning between authentic and artificially generated images holds significant importance, especially with the rise of AI-generated content. This project presents a novel approach to address this challenge through the development of a Convolutional Neural Network (CNN)-based image classifier.Leveraging TensorFlow's Keras API, our model achieves high precision and recall, in distinguishing between real and AI-generated images

Dataset

  • The dataset comprised of Real and AI Generated Images. Images were collected from Google using the python library pygoogle_image. Every single image was verified to avoid issues such as incorrect and ambigous images.
  • CIFAKE: Real and AI-Generated Synthetic Images

Combing the two datasets, we have:

  • Train Set: 101031
  • Test Set: 20000

Model

The model described is a convolutional neural network (CNN) developed using TensorFlow's Keras API, specifically designed for the binary classification of images into two categories: AI-generated and real images. It features an input layer for 32x32 RGB images, followed by multiple convolutional layers with ReLU activations and max pooling layers to extract and condense features. The network includes a flattening step to transform data for processing by fully connected dense layers, culminating in a sigmoid activation for probabilistic binary classification. The model uses the Adam optimizer and binary crossentropy loss, focusing on accuracy as its performance metric. This setup is optimized to differentiate complex patterns between AI-generated and authentic imagery effectively.


Testing and Evaluation

On evaluation, model achieved an accuracy of 93.90% Classification Report:

  • Precision: 0.92
  • Recall: 0.91
  • Accuracy: 0.91
